# 기능정리
* 병원정보 제공
    * 전체정보
    * 진료과목
    * 주소
    * 전화번호
    * 병원 종
    * 총 의료인 수
* 조건에 맞는 병원 리스트 출력
    * 지역에 해당되는 병원 
    * 지역 + 진료과목
    * 지역 + 병원 종
* 추가기능
    * 지역별 5대질병에 대한 경고도 및 대응법(실시간성)
    * corona 현황?(실시간성)
    * 네비게이션
    
# 시나리오

C - 안녕하세요 저는 경기도 지역별 병원정보 알리미입니다!

U - 의도4 도움말~  
C - 도움말 출력  
U - 김포시 병원 리스트!  (2)  
C - 병원리스트 출력  
C - 질병경고도 출력
U - 고려병원 전체정보 알려줘! OR 고려벙원 진료과목! OR 고려병원 위치알려줘!  (1)
C - 조건에 맞는 정보출력 및 길찾기(위치) 시 네비게이션기능 연결
