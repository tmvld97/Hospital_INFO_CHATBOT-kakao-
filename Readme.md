# 기능정리
* 병원정보 제공
    * 전체정보
    * 진료과목
    * 주소
    * 전화번호
    * 병원 종
    * 총 의료인 수
* 조건에 맞는 병원 리스트 출력
    * 지역에 해당되는 병원 
    * 지역 + 진료과목
    * 지역 + 병원 종
* 알람기능?
    * 지역별 5대질병에 대한 경고도 및 대응법
    * 의원에서 진료된 상위 4개 질병

# 개체명
**B_Hospital** : 병원<br/>
**B_Location** : 주소<br/>
**B_City** : 도시<br/>
**B_S_C** : 동<br/>
B_Tel : 전화번호<br/>
B_Subject : 진료과목<br/>
B_Treat : 진료과목<br/>
B_Type : 병원 종
